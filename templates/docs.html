What's Sublimall ?

Sublimall is a server and a SublimeText plugin.

The plugin create a compressed (and encrypted if you choose it) archive of your SublimeText configuration and send it to the server.

You can store for now one package at the same time.

After a long time you reinstall your system or you have a home and work computer so you can just type Ctrl-Shift-P and hit Sublimall retrieve to sync your other computer with your previously save configuration.

How to install SublimeText plugin ?

You'll need to have 7zip installed on your system.

  • Ubuntu: apt-get install p7zip-full
  • Osx: brew install p7zip
  • Windows: Go to 7zip website

In order to install Sublimall plugin you have two options, easiest method is if you already have Package Control which I heavily recommand you to use.

Hit Ctrl-Shift-P Install Package and type Sublimall to install it.

How to configure Sublimall plugin ?

Sublimall plugin configuration is very simple.

Warning! Sublimall is for now only compatible with SublimeText 3!

You just have to click on Preferences, Package Settings, Sublimall and Settings - User.

This is default configuration:

{
    ////////////////////////////
    //// Sublimall settings ////
    ////////////////////////////
    "api_root": "http://sublimall.org",
    "api_upload_url": "/api/upload/",
    "api_retrieve_url": "/api/retrieve/",
    "backup_directory_name": "backups",

    ///////////////////////
    //// User settings ////
    ///////////////////////
    "email": "",
    "api_key": "",
    // Set true if you want to exclude packages managed by Package Control
    "exclude_from_package_control": false,
    // Set false if you don't want to encrypt your configuration
    "encrypt": true,
    // Upload timeout in seconds
    "http_upload_timeout": 60
}

Add you email and your API key which you can find on your account page.

If encrypt option is true, you'll prompt to a passphrase when you'll upload your configuration on Sublimall.

Warning! This passphrase will be showed in cleared text in input bar. After you enter your passphrase, the plugin will encrypt your archive and send it to server.
